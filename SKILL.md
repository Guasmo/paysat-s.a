# Design Guidelines for PAYSAT S.A.

## Typography
- **Primary Font**: Inter (sans-serif)
- **Headings (h1-h6)**: bold, high contrast, clean spacing.
- **Body Text**: 16px base, 1.5 line height for readability.

## Color Palette
- **Primary Blue**: #1E88E5 (Used for primary actions like "Banca Virtual")
- **Background**: White/Light gray for clean contrast.
- **Text**: Dark Slate (#1A1A1A) for maximum readability.

## Component Principles
- **Hero**: Use a large bold title, clear subtitle, and prominent Call to Action.
- **Cards**: Soft borders, rounded corners (8px), subtle shadows.
- **Spacing**: Use standard 8px grid (8px, 16px, 24px, 32px, 48px).
